# Design-style images for the homepage

Copy images from `Project_Info/Design-Style-Images` into this folder.

Suggested filenames for the homepage layout:
- **hero.jpg** (or .png) – Hero / banner image
- **stock.jpg** – Stock items / immediate shipping
- **profiles.jpg** – Custom profiles / 40,000+ tooled
- **custom.jpg** – Replicate / precision installation

Any image format supported by the browser (jpg, png, webp) will work.
